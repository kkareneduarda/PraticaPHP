# Desafios de PHP

## Descrição
Consiste em cinco exercícios de lógicas em PHP: 
 - Desafio 1: Antecessor e Sucessor
 - Desafio 2: Sorteador de números
 - Desafio 3: Conversor de moedas simples
 - Desafio 4: Conversor de dólar consultando API
 - Desafio 5: Analisando um número Real

## Tecnologias Usadas
- PHP
- HTML

## Funcionalidades
- Antecessor e sucessor de um número
- Randomização
- Conversão
- Separação de parte inteira de fracionária


##  Executando o projeto
1 - Clone o repositório
2 - Coloque o projeto no htdocs
3 - Inicie o XAMPP
4 - Acesse `http://localhost/php`
